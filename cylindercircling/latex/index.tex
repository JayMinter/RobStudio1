\hypertarget{index_intro}{}\doxysection{Introduction}\label{index_intro}
This documentation provides an overview of the \mbox{\hyperlink{classCylinder}{Cylinder}} Detection Node, designed to detect cylindrical objects using laser scan data in ROS 2. The system processes scan data to find cylinder-\/shaped objects in the environment and visualizes them using RViz markers.\hypertarget{index_setup}{}\doxysection{Node Setup}\label{index_setup}
The system consists of the {\ttfamily \mbox{\hyperlink{classCylinder}{Cylinder}}} node, which subscribes to laser scan and odometry topics to detect and track cylinders.\hypertarget{index_cylinder_node}{}\doxysection{Cylinder Node}\label{index_cylinder_node}
The {\ttfamily \mbox{\hyperlink{classCylinder}{Cylinder}}} node subscribes to laser scan data ({\ttfamily /scan}) and odometry data ({\ttfamily /odom}). It processes the laser scan data to identify segments and detects cylinders by approximating circles using three points from a segment. Detected cylinders are visualized in RViz via markers.\hypertarget{index_cylinder_detection}{}\doxysection{Cylinder Detection Algorithm}\label{index_cylinder_detection}
The detection process follows these steps\+:
\begin{DoxyEnumerate}
\item {\bfseries{Segment Identification}}\+: Laser scan points are segmented based on proximity.
\item {\bfseries{\mbox{\hyperlink{classCylinder}{Cylinder}} Detection}}\+: Segments are analyzed by checking the curvature formed by three points, estimating a cylinder based on radius matching.
\item {\bfseries{Visualization}}\+: Once a cylinder is detected, it is visualized using {\ttfamily visualization\+\_\+msgs\+::\+Marker} in RViz.
\end{DoxyEnumerate}\hypertarget{index_behaviour}{}\doxysection{Cylinder Detection Behaviour}\label{index_behaviour}

\begin{DoxyEnumerate}
\item {\bfseries{Laser Scan Subscription}}\+: The node receives laser scan data and segments points based on proximity.
\item {\bfseries{Odometry Update}}\+: The odometry data is used to transform local laser scan points into global coordinates.
\item {\bfseries{\mbox{\hyperlink{classCylinder}{Cylinder}} Detection}}\+: The node checks for circular patterns in the segments and confirms cylinder detection by matching the radius.
\item {\bfseries{Visualization}}\+: Detected cylinders are visualized with cylinder markers in RViz.
